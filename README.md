# PhilipsHue

Toggle Lights On/Off in a Room

## Setup Virtual Env

`python -m venv venv`

`source venv/bin/activate`

## Install Project Requirements

`pip install -r requirements.txt`

## Setup App

`export FLASK_APP=hue`

## Start App

`flask run`
